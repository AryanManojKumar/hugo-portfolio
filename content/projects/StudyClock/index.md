**StudyClock** is a web-based productivity and study tracking application designed to help learners build consistent study habits through **time tracking, goal setting, and cloud-synced session history**.  
The application supports **Google authentication**, allowing users to securely access their study data across devices.

### Key Features
- **Google Authentication:** Secure sign-in using Google OAuth for personalized user sessions  
- **Cloud Sync:** Study sessions and progress stored in **Firebase Firestore**, accessible across devices  
- **Study Timer:** Start, pause, and track focused study sessions in real time  
- **Daily Goals:** Set daily study targets and visualize progress toward completion  
- **Session History:** Persistent log of study sessions with timestamps  
- **Live Clock UI:** Real-time clock display to reinforce time awareness and discipline
