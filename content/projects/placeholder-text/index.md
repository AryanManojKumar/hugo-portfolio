## Gym Buddy — AI-Assisted Fitness & Workout Recommendation App  
*Final Year B.Tech Project *

**Gym Buddy** is a **cross-platform mobile fitness application** built using **React Native**, designed to help users plan workouts, track fitness activity, and receive **AI-assisted exercise recommendations** based on selected muscle groups and available equipment.  
The app integrates **LLM-powered guidance** with **cloud-backed data synchronization** to deliver a personalized and accessible fitness experience.


---

### Key Capabilities

**Personalized Workout Recommendations**
- Recommends exercises based on **target muscle groups** and **available gym equipment**
- Designed to support beginners and intermediate users without predefined workout plans  

**AI-Powered Fitness Assistant**
- Integrated **Google Gemini API** to provide an AI chatbot for workout guidance  
- Answers exercise-related questions and suggests variations and form tips  

**Workout & Activity Tracking**
- Logs workouts and basic fitness activity  
- Tracks calories and exercise history over time  

**Cloud-Based Authentication & Sync**
- User authentication and session management  
- Real-time data storage and synchronization across devices using **Supabase**  

**Cross-Platform Mobile Experience**
- Built with **React Native** for Android and iOS compatibility  
- Mobile-first UI focused on ease of use inside gym environments  

---

### System Architecture

- **Mobile Client:** React Native app handling UI, navigation, and user interactions  
- **AI Layer:** LLM-powered assistant for exercise guidance and recommendations  
- **Backend / Data Layer:** Cloud database for storing user profiles, workouts, and activity logs  
- **Auth Layer:** Secure user authentication with per-user data isolation  

The architecture separates **UI logic**, **AI reasoning**, and **data persistence**, allowing each component to evolve independently.

---

### Tech Stack

- **Mobile App:** React Native  
- **AI Integration:** Google Gemini API  
- **Backend & Auth:** Supabase  
- **Database:** Supabase (real-time data storage)  
- **Language:** JavaScript
